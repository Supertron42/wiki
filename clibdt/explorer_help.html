Explorer Overview

The Explorer tool provides quick access to all ClibDT-related folders and directories.

Features

  • Quick access to development folders
  • Double-click to open in Windows Explorer
  • Visual indicators for existing/missing folders
  • Tooltips showing full paths and descriptions

Folder List

The Explorer shows these ClibDT folders:

FolderIconDescription
Development Root🏠Main ClibDT development directory
Projects📁All your ClibDT projects
Builds📦Compiled plugin outputs (DLL files)
Tools🔧Development tools and utilities
Downloads⬇️Downloaded files and archives
Resources📚Headers, libraries, CommonLibSSE
Backups💾Automatic project backups
Config⚙️ClibDT configuration files

Using Explorer

Opening Folders

  1. Go to Explorer in the left panel
  2. The folder list shows all ClibDT directories
  3. Double-click any folder to open it in Windows Explorer

Folder Status

  • Existing folders are shown normally
  • Missing folders show a warning in the tooltip
  • Hover over any folder to see its full path
Missing Folders: If a folder doesn't exist, ClibDT will open the nearest existing parent folder instead.

Quick Navigation

Use Explorer to quickly jump to:

  • Projects - to open your source code in an editor
  • Builds - to find compiled DLL files
  • Config - to view or edit settings files
  • Backups - to restore previous versions
Tip: Keep Explorer open while working to quickly access any folder without navigating through Windows Explorer.
