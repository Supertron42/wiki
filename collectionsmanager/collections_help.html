Best Practices

Purpose of the Collections Tab

The Collections tab shows all your saved collections and provides quick access to manage them. From here you can:

  • Load a previously saved collection to continue installation
  • View collection progress and completion status
  • Access the collection on Nexus Mods
  • Uninstall or delete collections
  • Organize collections by reordering them
  • Check for collection updates
Tip: Collections are automatically saved when you load them from a Nexus URL. They persist between MO2 sessions and can be reloaded at any time.

Collection Cards

Each saved collection is displayed as a card showing key information at a glance:

  • Thumbnail - The collection's image from Nexus Mods
  • Name - The collection title
  • Author - Who created the collection
  • Progress - How many mods are installed vs total
  • Revision - The collection version number
  • Action buttons - Load, View, Copy URL, Uninstall, Delete

Thumbnail & Info

The left side of each card shows:

  • Reorder arrows - Move collection up or down in the list
  • Thumbnail image - Loaded from Nexus Mods (9:16 aspect ratio)

The middle section displays:

  • Collection name - Bold title text
  • Author - "by AuthorName" in muted text
  • View/Copy buttons - Quick access to Nexus URL
  • Progress stats - Installed count, skipped count, revision

Progress Tracking

The progress line shows your installation status:

Progress 45/120 (Installed 42, Skipped 3) | Rev 15
  • Progress X/Y - Done count vs total mods in collection
  • Installed - Number of mods successfully installed
  • Skipped - Mods you chose to skip (counts as "done")
  • Rev - Collection revision number from Nexus
Done = Installed + Skipped. A collection is "complete" when Done equals Total, regardless of how many were skipped.

LOCAL Collections

Collections marked LOCAL were imported from a local file (like a mod list export). These have limited functionality:

  • No Nexus URL available
  • View and Copy URL buttons are disabled
  • Cannot check for updates

Loading Collections

Click the Load button on any collection card to load it:

  1. The Mods tab populates with the collection's mod list
  2. Status of each mod is detected (installed, downloaded, pending)
  3. You can continue downloading and installing from where you left off
Resume anytime: Your progress is saved automatically. Load a collection days or weeks later and pick up where you stopped.

View & Copy URL

ButtonAction
ViewOpens the collection page on Nexus Mods in your browser
Copy URLCopies the collection URL to your clipboard for sharing

These buttons are disabled for LOCAL collections that don't have a Nexus URL.

Uninstalling Collections

Click Uninstall to remove installed mods but keep downloads:

  • All mods from this collection are removed from your MO2 mods folder
  • Downloaded archives are kept in your downloads folder
  • The collection remains in your saved list
  • You can reinstall later without re-downloading
Note: This removes the mod folders. Any manual changes you made to installed mods will be lost.

Deleting Collections

Click Delete (red text) to completely remove a collection:

  • All mods from this collection are uninstalled
  • Downloaded archives are deleted
  • The collection is removed from your saved list
  • This action cannot be undone
Caution: Delete permanently removes both mods and downloads. Use Uninstall if you want to keep the downloaded archives.

Reordering Collections

Use the arrow buttons on the left side of each card to reorder your collections:

  • Up arrow - Move collection up in the list
  • Down arrow - Move collection down in the list

The order is saved automatically and persists between sessions.

Program Installation

An orange banner appears when mods requiring manual game folder installation are detected. Click Install to Game Folder to manage them:

  • Script extenders (SKSE, F4SE, OBSE, etc.)
  • ENB and ReShade files
  • Engine-level fixes and patches
  • Any files that can't work through MO2's virtual file system

Two Tabs

  • Install to Game Folder - View detected root mods, open their folders, and mark them as done after copying files
  • Install Rules - Configure which mods should always or never be flagged for root installation

Tracking Progress

After manually installing files, click the Done button on each mod. The banner count updates and hides when all mods are handled.

Check for Updates

Click Check for Updates to see if any of your saved collections have new revisions on Nexus Mods:

  1. Each collection's current revision is compared to Nexus
  2. Collections with updates available are highlighted
  3. You can choose to update to get new mod additions or changes
Tip: Collection authors may release updates to add new mods, update versions, fix issues, or improve compatibility.

Refresh

Click Refresh to reload the collections list:

  • Re-reads saved collection data
  • Updates progress counts
  • Reflects any changes made in other tabs

Best Practices

  • One collection at a time - Focus on completing one collection before starting another
  • Check progress regularly - The progress indicator shows how close you are to completion
  • Use Uninstall for testing - Uninstall keeps downloads so you can quickly reinstall
  • Organize by priority - Reorder collections to put your most-used ones at the top
  • Check for updates - Collection authors often release improvements
  • Back up before delete - Once deleted, you'll need to re-download everything

When to Load vs Start Fresh

  • Load existing - Continue from where you left off, keep your progress
  • Uninstall + Load - Start fresh but keep downloads
  • Delete + Reload URL - Complete fresh start, re-download everything

Managing Multiple Collections

If you have mods from multiple collections:

  • Collections can share mods - duplicates are detected automatically
  • Use the Duplicate Finder tool to clean up overlapping mods
  • Load order matters - check the Sort tool after switching collections
