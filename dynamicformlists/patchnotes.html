Version 1.0.0 - Initial Release

Features

  • NEW Restores missing form list entries. When multiple mods edit the same form list, entries added by overridden mods are automatically restored at startup.
  • NEW Respects intentional removals. If a mod specifically removed an entry from a form list, it stays removed.
  • NEW xEdit companion script. Added ST - Dynamic Form Lists.pas for SSEEdit. Info Mode checks for broken references, null FormIDs, duplicates, and empty lists. Patch Mode creates a merged ESP.
  • NEW Skip specific mods via INI. Add plugin filenames to the skipMods setting to exclude them from processing.
  • NEW Skip specific form lists via INI. Add EditorIDs to the skipFormLists setting to leave certain lists untouched.
  • NEW Detailed optional logging. Turn on enablelogs to see exactly which entries were restored for each form list.
  • NEW SE, AE, and VR support. Single plugin binary covers all Skyrim editions via CommonLibSSE-NG.
