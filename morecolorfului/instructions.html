Getting Started

Requirements

  • Skyrim SE (1.5.97 or 1.6.x), AE, or VR
  • SKSE64 (or SKSEVR for VR) - Download here
  • Address Library for SKSE Plugins - Nexus page

Installation

Using a Mod Manager (Recommended)

  1. Download the mod from Nexus
  2. Install through MO2, Vortex, or your preferred mod manager
  3. Enable the mod
  4. Configure MoreColorfulUI.ini to your liking

Manual Installation

  1. Extract the archive
  2. Copy MoreColorfulUI.dll to Data\SKSE\Plugins\
  3. Copy MoreColorfulUI.ini to Data\SKSE\Plugins\

File Structure

Data/
  SKSE/
    Plugins/
      MoreColorfulUI.dll
      MoreColorfulUI.ini

Quick Configuration

Open MoreColorfulUI.ini and add color mappings under [Colors]:

[Colors]
Take=#00FF00
Open=#FFD700
Steal=#FF0000
Read=#87CEEB
Activate=#FFFFFF

Colors are applied based on the prefix of the activation text. See INI Settings for full details.

Note: Changes to the INI file require a game restart to take effect.

Troubleshooting

Plugin not loading

  • Verify SKSE is installed correctly by checking the main menu (should show SKSE version)
  • Make sure Address Library is installed for your game version
  • Check Documents\My Games\Skyrim Special Edition\SKSE\MoreColorfulUI.log for errors

Colors not applying

  • Ensure your prefix matches the start of the activation text exactly (case-sensitive)
  • Use valid hex color codes (e.g., #FF0000 or FF0000)
  • Restart the game after making INI changes
