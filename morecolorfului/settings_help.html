INI Settings

Configure More Colorful UI by editing Data\SKSE\Plugins\MoreColorfulUI.ini

File Location

The INI file should be located at:

Data\SKSE\Plugins\MoreColorfulUI.ini

If using a mod manager, edit the file within your mod's folder structure.

Basic Format

[Colors]
Prefix=HexColor

Each line under [Colors] maps a text prefix to a color. When activation text starts with that prefix, it will be displayed in the specified color.

How Prefix Matching Works

The plugin checks if the activation text starts with any configured prefix. The first matching prefix wins.

Important: Matching is case-sensitive. Take will match "Take Iron Sword" but not "take iron sword".

Example Configuration

[Colors]
; Interaction types
Take=#00FF00
Open=#FFD700
Steal=#FF0000
Read=#87CEEB
Sleep=#9B59B6
Sit=#808080

; Specific items (more specific prefixes should come first)
Take Gold=#FFD700
Take Potion=#FF69B4

Color Format

Colors should be specified as hex color codes:

  • #RRGGBB format (with hash)
  • RRGGBB format (without hash)

Common Activation Prefixes

Prefix Usage Example Color
Take Picking up items #00FF00
Open Doors, containers #FFD700
Steal Owned items #FF0000
Read Books, notes #87CEEB
Talk NPCs #98FB98
Activate Generic interactions #FFFFFF
Search Dead bodies, containers #DDA0DD
Harvest Plants, ingredients #32CD32
Unlock Locked items (with key) #C0C0C0
Sleep Beds #9B59B6

Tips

  • More specific prefixes should be listed before general ones (e.g., "Take Gold" before "Take")
  • Use comments (lines starting with ;) to organize your configuration
  • Test colors in-game to ensure readability against different backgrounds
  • Restart the game after making changes to the INI file
