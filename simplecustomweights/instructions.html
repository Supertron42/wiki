Getting Started

This guide walks you through installing Simple Custom Weights and verifying it's working correctly.

Installation

1

Install with your mod manager — Install the mod like any other SKSE plugin. MO2, Vortex, or manual installation all work.

2

Verify the DLL location — Confirm the DLL is at: Data/SKSE/Plugins/SimpleCustomWeights.dll

3

Create or edit the INI — The settings file goes at: Data/SKSE/Plugins/SimpleCustomWeights.ini

Tip: If the INI doesn't exist, the plugin will use default values. You only need to create it if you want to customize weights.

First Run Verification

  1. Launch the game to the main menu, then load a save or start a new game.
  2. Open your SKSE log directory (usually Documents/My Games/Skyrim Special Edition/SKSE).
  3. Look for SimpleCustomWeights.log and open it.
  4. You should see:
    • INI load attempt and the settings that were read
    • Patch summaries like "Patched X weapons", "Patched X armors", etc.

When Weights Are Applied

  • On initial load: Patches apply when the DataLoaded message fires
  • On save load: Patches reapply on PostLoadGame
  • On demand: Use the Papyrus reload function to apply changes without reloading

Tip: If you edit the INI while the game is running, just load a save and your changes will take effect — no game restart needed.

Recommended Workflow

  • Start small: Make small adjustments first. Drastically reducing weights can trivialize carrying capacity and change gameplay balance.
  • Keep zero-weight protection on: Leave ApplyMinimumWeightToZero=true unless you have a specific reason to allow true zero weights.
  • Test your changes: After editing the INI, reload weights and check a few items in your inventory to confirm the changes.

Note: Quest items and items with the VendorNoSale keyword are always skipped to prevent breaking quests or game mechanics.
