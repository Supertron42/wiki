1.0.0 Initial Release

Features

  • Runtime weight patching for weapons, armor, potions, scrolls, food, books, ingredients, soul gems, misc items, and tools
  • Dedicated gold weight override
  • Optional fixed weights for keys and lockpicks
  • Automatic reload on save load
  • Papyrus function for manual reload
  • Zero-weight protection with configurable minimum
  • Quest item and VendorNoSale blacklist
